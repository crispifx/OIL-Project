# OIL-Project
Repository for the OIL Project

O.I.L. (Ojos Intérpretes del Lenguaje)
O.I.L. is a wearable technology project featuring smart glasses that use AI to provide real-time speech translation.

Overview
The device listens to different languages through a built-in microphone and displays the translated text directly onto the lenses, enabling seamless communication between people speaking different languages.

Key Features
Real-time AI Translation: Automatic speech recognition and translation.

Smart Display: Text is projected directly onto the glass lenses.

Portable & Durable: Designed to be lightweight and comfortable for daily use.
